# FFmpeg GUI

A Small, but handy GUI for the Windows version of FFmpeg. It's currently in development, but if you see potential in it feel free to clone & contribute

## Features

* Preset based
* Easy to extend
* Wizard Like

## Requirements to run

* .NET Core 3.1

## Howto get a build

```
git clone https://github.com/webmaster442/FFmpegGui.git
cd FFmpegGui
dotnet build
```
